Time_off_requests

  • profile_id : String
    ID of the user related to the time off request.
  • leave_start : String
    Date that the time off request begins.
  • leave_end : String
    Date that the time off request ends.
  • requester_id : String
    The ID of the user submitting the time off request.
  • time_off_type_id : String
    The ID of the time off type. (E.g. PTO ID, Sick ID, etc.)
