This is the API

profilesFieldsGet

Returns all currently supported Fields, default and custom. NAME is the internal name of the field, which is also the key that should be used in the profile API endpoints. This may be distinct from LABEL, which is what will appear in the web interface. The valid format info is an attempt to explain what the proper format and possible values are for that field.

URL

https://company.namely.com/api/{version}/profiles/fields

HTTP Method

GET

Response Type

inline_response_200_5

Parameters

profilesFieldsPost

Allows you to create a profile field within a section.

URL

https://company.namely.com/api/{version}/profiles/fields

HTTP Method

POST

Response Type

inline_response_200_5

Parameters

  • Query: section_id String

    Section ID

  • Query: title String

    Field Title

  • Query: type String

    Field Type

  • Query: choices String

    An array of choices for checkboxes and select fields.
